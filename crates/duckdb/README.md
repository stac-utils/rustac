Experimental client using [DuckDB](https://duckdb.org/) to search [stac-geoparquet](https://github.com/stac-utils/stac-geoparquet).

## Usage

In your `Cargo.toml`:

```toml
[dependencies]
stac-duckdb = "0.0.2"
```

See the [documentation](https://docs.rs/stac-duckdb) for more.
